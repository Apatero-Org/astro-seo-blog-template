# 新しいブログへようこそ！

新しい Astro SEO ブログテンプレートのセットアップおめでとうございます！この例示投稿は、自分のブログ投稿を作成し、構造化する方法を理解するのに役立ちます。

## このテンプレートとは？

これは Astro で構築された、モダンで機能豊富なブログテンプレートです。以下が含まれます：

- **SEO 最適化**：検索エンジン向けの組み込みベストプラクティス
- **ダークモード**：自動テーマ切り替え
- **多言語サポート**：国際化対応
- **高速パフォーマンス**：超高速ページ読み込み
- **簡単なカスタマイズ**：シンプルな設定とスタイリング

## 最初の投稿を作成する

ブログ投稿は MDX 形式で書かれており、Markdown と React コンポーネントの両方を使用できます。`public/data/posts/` に `.mdx` 拡張子の新しいファイルを作成します。

### フロントマター

すべての投稿は、メタデータを定義するフロントマター（`---` マーカー間のセクション）で始まります：

```yaml
---
title: "投稿のタイトル"
description: "簡単な説明"
pubDate: "2025-01-15"
author: "あなたの名前"
category: "技術"
tags: ["astro", "web開発"]
image: "/blog-images/your-image.jpg"
featured: false
---
```

### コンテンツの作成

フロントマターの後、Markdown を使用してコンテンツを書きます：

- **太字テキスト**は `**text**` で
- *斜体テキスト*は `*text*` で
- [リンク](https://astro.build)は `[text](url)` で
- コードブロックは三重バッククォートで

### コード例

シンタックスハイライト付きのコードを含めることができます：

```javascript
function greet(name) {
  console.log(`こんにちは、${name}！`);
}

greet("世界");
```

```python
def greet(name):
    print(f"こんにちは、{name}！")

greet("世界")
```

## コンテンツの整理

### カテゴリ

カテゴリ別に投稿を整理します。`public/data/categories/` でカテゴリを定義します。

### タグ

より良い発見可能性のために、各投稿に複数のタグを追加します。`public/data/tags/` でタグを定義します。

### 画像

`public/blog-images/` に画像を保存し、投稿で参照します：

```markdown
![代替テキスト](/blog-images/your-image.jpg)
```

## ブログのカスタマイズ

### サイト設定

`public/data/settings/site-config.json` を編集してカスタマイズします：

- サイトのタイトルと説明
- 著者情報
- ソーシャルメディアリンク
- ナビゲーション項目
- 機能（ダークモード、検索、RSS）
- アナリティクストラッキング

### スタイリング

テンプレートは Tailwind CSS を使用します。`tailwind.config.mjs` で色、フォント、スタイルをカスタマイズします。

### コンポーネント

`src/components/` でカスタムコンポーネントを作成し、MDX 投稿で使用します。
